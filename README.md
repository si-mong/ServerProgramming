# 서버프로그래밍 (Server Programming)

---

# 1. 한국어 버전 (Korean Version)

## 1. 강의개요  
이 교과목은 서버 프로그래밍의 기본 개념부터 실제 서비스 개발에 필요한 기술까지 폭넓게 다루는 수업입니다.  
Linux 기반 환경에서 C 언어를 중심으로 서버 프로그램을 작성하며, 이론과 실습을 병행하여 실무와 가까운 경험을 제공합니다.

## 2. 학습목표  
- Linux 시스템 기반 서버 프로그래밍의 원리를 이해한다.  
- 파일 입출력, 프로세스, 스레드, 소켓 프로그래밍 등 핵심 기술을 학습한다.  
- 다양한 실습과 프로젝트를 통해 서버 프로그램 구현 역량을 강화한다.  

## 3. 주차별 내용  
1주차: 운영체제 기초 복습  
2주차: 기본 명령어 실습  
3주차: 개발 환경 구성 1  
4주차: 개발 환경 구성 2  
5주차: 파일 입출력 1  
6주차: 파일 입출력 2  
7주차: 파일 시스템  
8주차: 프로세스 1  
9주차: 프로세스 2  
10주차: 스레드  
11주차: 소켓 프로그래밍 1  
12주차: 소켓 프로그래밍 2  

## 4. 개인 과제  
- Micro Project: Web Server (C 기반)  
- Micro Project: Django Server  
- Micro Project: Node.js Server  
- Micro Project: Spring Framework Server

# 2. English Version

## 1. Course Overview  
This course provides a comprehensive introduction to server programming, covering both fundamental theories and practical skills.  
Students will work in a Linux environment, primarily using the C programming language, and gain hands-on experience through exercises and micro projects.

## 2. Learning Objectives  
- Understand principles of server programming on Linux systems.  
- Learn key topics such as file I/O, processes, threads, and socket programming.  
- Strengthen implementation skills through practical exercises and micro projects.  

## 3. Weekly Topics  
Week 1: Operating System review  
Week 2: Basic commands  
Week 3: Development environment setup 1  
Week 4: Development environment setup 2  
Week 5: File I/O 1  
Week 6: File I/O 2  
Week 7: File system  
Week 8: Process 1  
Week 9: Process 2  
Week 10: Thread  
Week 11: Socket programming 1  
Week 12: Socket programming 2  

## 4. Individual Micro Projects  
- Micro Project: Web Server (C-based)  
- Micro Project: Django Server  
- Micro Project: Node.js Server  
- Micro Project: Spring Framework Server
